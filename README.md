Gogo Sushi | Cihuatlán - Menú Digital

¡Bienvenido al repositorio de Gogo Sushi! Este es un proyecto de menú digital interactivo diseñado para ofrecer una experiencia visual atractiva y moderna a los clientes de la sucursal en Cihuatlán, Jalisco.

Vista Previa
El sitio está optimizado para dispositivos móviles y escritorio, permitiendo a los usuarios navegar rápidamente entre las diferentes categorías de comida japonesa y snacks.

Características
* Diseño Responsive: Adaptado para celulares, tablets y computadoras.
* Navegación Intuitiva: Menú pegajoso (*sticky*) para acceso rápido a las secciones.
* Categorías Detalladas: Secciones específicas para Rollos, Yakimeshi, Teppanyaki, Hamburguesas, Brochetas y Acompañamientos.
* Páginas Secundarias: Archivos dedicados para Bebidas y Extras para no saturar la vista principal.
* Estética Moderna: Uso de sombras, gradientes y tipografías personalizadas (Montserrat & Noto Sans).

Tecnologías Utilizadas
* HTML5: Estructura semántica del contenido.
* CSS3: Estilos personalizados, Flexbox, CSS Grid y animaciones de transición.
* Boxicons: Para la iconografía del sitio.
* Google Fonts: Fuentes integradas para mejorar la legibilidad.

Estructura del Proyecto
```text
├── IMG/                  # Carpeta con las imágenes de los platillos y logos
├── Gogo_Sushi.html       # Archivo principal (Menú de comida)
├── bebidas.html          # Sección de bebidas
├── extras.html           # Sección de complementos y salsas
├── style.css             # Estilos generales del proyecto
└── README.md             # Documentación del proyecto
```
